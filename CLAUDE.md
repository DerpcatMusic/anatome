## Design Context

### Users
HomeBody serves Hebrew-speaking Pilates members and instructors using the product from home. Members need calm, clear access to live classes, video lessons, and profile setup; instructors need reliable studio controls that feel technical enough to trust during a session without becoming cluttered.

### Brand Personality
Quiet, exacting, warm. The interface should feel like a polished independent video tool for embodied practice: confident, slightly experimental, and performance-aware.

### Aesthetic Direction
The visual direction is pre-internet meets postmodern video infrastructure: hard edges, visible borders, large confident Hebrew type, monospace labels, mesh gradients, blur, and occasional pill controls. Mux is the strongest reference for alignment and product feel: crisp grids, dense technical modules, playful but disciplined UI, and gradients used as environmental texture. Avoid raw browser-form sprawl, uneven box sizing, generic SaaS cards, and a flat beige-only palette.

### Design Principles
Use strict alignment and repeated dimensions so controls feel engineered, not accidental.
Pair hard rectangular structure with soft mesh/blur surfaces and a few pill accents for contrast.
Make live/instructor surfaces feel operational and premium: compact controls, visible status, strong hierarchy, minimal explanation text.
Preserve RTL ergonomics while keeping technical labels and values easy to scan.
Prefer polished native-feeling controls over decorative excess; every visual layer should clarify state or hierarchy.
